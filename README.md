# highfence devlog

Personal developer blog published at <https://highfence.github.io>.

This site is a Jekyll-based GitHub Pages devlog. Draft writing can still happen in the local Obsidian vault, but only reviewed public posts should be copied into `_posts`.

## Local Commands

```powershell
bundle install
bundle exec jekyll serve
```

## Publishing Rule

Do not copy the whole Obsidian vault into this repository. Review every public post for company-internal details, private project names, logs, screenshots, credentials, and personal identifiers before publishing.

GitHub Pages builds this repository with Jekyll. Do not add `.nojekyll`.
